Política de Privacidade — Solicitar Ativos (Magalu) Esta extensão foi desenvolvida para uso interno na Magazine Luiza e não coleta, armazena, transmite nem compartilha dados pessoais de nenhum usuário com terceiros. O token de autenticação (JWT) interceptado pela extensão é utilizado exclusivamente para realizar requisições autenticadas à API interna do portal gestaoativos.magazineluiza.com.br, em nome do próprio usuário logado. Este token permanece apenas na memória do navegador durante a sessão ativa e nunca é enviado a servidores externos ou armazenado localmente. Nenhuma informação é coletada, registrada ou transmitida para fora do domínio gestaoativos.magazineluiza.com.br. Dúvidas: joao.gmarques@magazineluiza.com.br
