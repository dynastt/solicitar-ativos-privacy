Política de Privacidade

Solicitar Ativos — Magalu  ·  Última atualização: março de 2026

1. Sobre esta extensão

A extensão Solicitar Ativos — Magalu foi desenvolvida para uso interno na Magazine Luiza. Seu único propósito é automatizar solicitações de ativos em lote no portal interno gestaoativos.magazineluiza.com.br.

2. Dados coletados

Esta extensão não coleta, não armazena e não transmite nenhum dado pessoal a servidores externos ou terceiros.

O token de autenticação (JWT) interceptado pela extensão é utilizado exclusivamente para realizar requisições autenticadas à API interna do portal, em nome do próprio usuário já logado. Este token permanece apenas na memória do navegador durante a sessão ativa e nunca é gravado em disco ou enviado para fora do domínio gestaoativos.magazineluiza.com.br.

3. Permissões utilizadas

activeTab: utilizada para injetar o painel de interface na aba ativa do portal interno e interceptar os tokens de autorização das requisições da página.

Host permission (gestaoativos.magazineluiza.com.br): necessária para que os scripts de conteúdo funcionem exclusivamente neste domínio. Nenhum outro site é acessado.

4. Compartilhamento de dados

Nenhuma informação é compartilhada com terceiros. A extensão não possui servidor próprio, não usa analytics e não exibe anúncios.
